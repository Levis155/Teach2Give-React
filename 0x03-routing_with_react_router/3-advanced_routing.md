# Advanced Routing

There is a lot we can do with React Router to make more complex routes easier to read and overall much more functional.

This can be done using the following techniques:

- Dynamic Routing

- Routing Priority

- Nested Routes

- Multiple Routes

- _useRoutes_ hook
